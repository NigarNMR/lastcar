Если ранее рассчет % не производился, то необходимо нажать на кнопку "Расчет соответствий".
Для того, чтобы попасть на страницу "Таблица брендов по степени соответствия группам", необходимо зайти на страницу "Управление группами брендов" и нажать на кнопку "Страница соответствий".

Для того, чтобы попасть на страницу "Дубликаты брендов", необходимо зайти на страницу "Управление группами брендов" и нажать на кнопку "Страница дубликатов".

Для того, чтобы попасть на страницу "Проверки методов web-поставщиков", необходимо кликнуть на кнопку "WS" на панели администратора.

Для включения класса логирования, в настройках tecdoc модуля необходимо передать параметру LOGGING_ENABLED значение TRUE.

Для того, чтобы попасть на страницу "Управление брендами WS / Каталогов", необходимо кликнуть на кнопку "Словари" на панели администратора.
